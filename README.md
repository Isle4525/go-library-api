# LibaryBookControl

Система управления библиотечным фондом на языке Go. Проект находится в стадии активной **разработки (Work In Progress)**.

## 📌 О проекте

Данное приложение предназначено для автоматизации учета книг, управления базой читателей и отслеживания истории выдачи литературы. Архитектура построена на принципах разделения ответственности (Layered Architecture), что обеспечивает легкость тестирования и расширения.

## 🛠 Технологический стек

- **Language:** Go (Golang)
- **Database:** PostgreSQL (используется `database/sql` со стандартными драйверами)
- **Architecture:** Repository Pattern + Service Layer

## 🏗 Структура проекта

На данный момент реализованы следующие слои:

- `internal/models/` — описание сущностей (Книги, Авторы, Пользователи, Записи о выдаче).
- `internal/repository/` — низкоуровневая работа с базой данных (CRUD операции).
- `internal/services/` — бизнес-логика приложения (валидация, агрегация данных).

## ⚙️ Как запустить (в разработке)

> **Примечание:** Так как проект находится в разработке, точка входа (`main.go`) может отсутствовать или быть в процессе настройки.

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/your-repo/LibaryBookControl.git
   ```
2. Настройте переменные окружения для подключения к PostgreSQL.
3. Установите зависимости:
   ```bash
   go mod tidy
   ```
